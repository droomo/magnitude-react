## 图形绘制

使用[Konva](https://konvajs.org/)作为`canvas`的扩展绘图库。

## 文件解释

- `KonvaPatch.ts`文件包含实验需要的`缺口圆`和`成对缺口圆`的绘制过程，作为`konva`库的扩展图形。
- `StimuliGenerator2.tsx`是刺激生成过程的文件。
- `StimuliViewer.tsx`是预览已生成刺激（及生产图片）的文件。
